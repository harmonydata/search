The syntax runs on CAMS data to create derived variables denoting persons in households which contained one or more migrants. Separate sets of these variables may be derived for relationships to all persons in the household, relationships to other migrants, and relationships to other migrants in the same 'moving group'. Within the original project, the derived variables were constructed for migrants only (about 11% of all CAMS records).
